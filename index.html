Word Design

At the beginning of the semester the class learned about typography and the importance of it. We learned the components of typeface, and the various styles that come with it. We were taught the vocabulary of fonts and the impact it has on someone when reading text.

Type Design

Next in the semester we learned about Type Design. During that unit we were taught the importance of spacing and using empty space. Good alignment was needed while still not being to close. During this project the class also learned the importance of colour, using only black and another colour.

Self Portrait

For this project we replicated a digitally edited photo, we had many options to chose from and for this project I decided to replicate an image of me holding a mask of my own face. I greatly enjoyed this project and liked the options we had to chose from.

Sound Track

The next project the class worked on was creating a sound track. We used Mixcraft which had a variety of different instruments and tracks that we could use.

Production Logo

In this project we designed a personal logo that had our name, and we then turned it into an animation.

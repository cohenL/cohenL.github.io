Website Design

Website Design

The most recent project that the class has been given was to design a website to be a portfolio of all of our previous projects we have worked on thought the semester. The website is supposed to have multiple pages with various subjects such as a home page, contact me page and about me page.

Initial Thoughts

I knew that I wanted my design for the website to be unique and stand out from others. My initial thoughts was to have four different tabs, and have the home page to direct the user to those tabs. I also wanted one of the tabs to direct the user to a sub page for all of my projects.

Collapsible Text

I wanted to use different aspects of google sites to make each page of the website slightly different than the other. I planned to have collapsible text for my "my projects" page to display all of the projects with a small thumbnail picture of one of the projects. A problem that I ran into is that the collapsible text did not allow images to also be in the collapsible portion. This prevented me from completing what I envisioned.

Embedding Problem

I found a solution to the problem by using the embed tag and write html code to make the collapsible text. After I wrote the code, it functioned to a certain extent. I could not insert a link to another page using the embed tag. Instead it would open a new chrome tab and ask for the user to sign in. This is not what I intended for the my projects page to look like, because I wanted it to be able to link it to the projects it was displaying.

Image Carousel

For my home page I hoped I would be able to make an image carousel to hold the contents of all of my pages. The idea I envisioned was to have 3 different slides showcasing the 3 others pages, My Projects, About me and Contact me. One problem I encountered was that the carousel could only have images inside it and, it can't link to other pages.

Embedding Problem

I attempted to use the embed tag again, and write html code for the carousel, but I once again was not able to link to other pages. The whole entire idea of the home page was to direct te user to the other pages, with a brief description of what each page had to show.

Limitation of Sites

Many of my envisioned ideas weren't possible using google sites and that caused it to less unique than I planned. Google sites is also much more limited for customization, you are really restricted when using the google sites and can't have as much creativity. Photos aren't able to be in the exact sizes you want, and font and text size is limited to the themes chosen.

Using HTML

I have been taking a online class about web development, and have learned how to create websites using html, stylesheets and javascript and take advantage of the bootstrap libraries. I was able to replicate the google sites look using bootstrap, and was now able to incorporate my ideas.

Collapsible Text

The first component I created that was unique to my html website was creating the collapsible text. I was able to use the bootstrap accordion and customize it to have text, images and links to other pages. The accordion design was to only have one subject open at a time, and if you open a new one it closes the previous. This prevents clutter and helped keep the page look better.

Image Carousel

Next, I used the carousel from bootstrap in my other website. This carousel allowed text to be in it, so I was able to have a small description of each page and I was able to link to the pages I was describing. This is the design I first had planned, the home page having a thumbnail picture of the pages, a small description and an easy way to go the pages. Compared to the google sites carousel that only allowed images and it could not link to other pages

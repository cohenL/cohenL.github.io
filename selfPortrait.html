Self Portrait

Next the class started to learn how to use photoshop, and the project was to follow a design and create one of our own. There were various options for the class to recreate and all had different levels of difficulty. Initially I decided to recreate an image with a glossy style and painted effect. When I attempted at the project I realized that to create the effect it was very time consuming and difficult to do. We were only given a limited amount of days to do the project so I decided to change which design I was going to recreate. The next project I decided to replicate was one with myself removing a mask of my own face.

Self Portrait

This was under the same difficulty of the previous project, so it would be worth more points when graded. When working on the project I found it much easier to complete, the first part of the project was to create the mask. I created a new layer and duplicated a outline of my face, then I had to tilt it along with my arm and hand. Then I made the edges of the mask to look more like plastic and created the string that was going around my head. Next I had to remove the previous arm and face and add the shadow of the mask. Afterwards I had to create the background using two different gradients and the project was complete.

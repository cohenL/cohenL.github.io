Word Design

At the beginning of the semester the class learned about typography and the impact it has on text. The class learned the components of typeface, and the various styles that come with it. We were taught the vocabulary of fonts and the impact it has on someone when reading text. The first project we had to do was using only the words "Com Tech" we had to create 10 different designs using Sans-Serif typeface.

Word Design

This was a challenging project, due to it being the first project we have ever done and the only guideline we were given was have the words Com Tech. The project was solely based on creativity and having a open mind. There were two parts to the project, the first page was 10 designs using only black and white and the second page as using any colours we wanted. This let more creativity to be aloud due to being able to add more to the designs using colours.
